# Tema-Integrador-I
# 💻 Projeto Integrador - Ciência de Dados

Este projeto foi desenvolvido como parte de uma atividade acadêmica com o objetivo de explorar a área de Ciência de Dados, compreender o uso do GitHub e criar um site profissional.

## 🚀 Sobre o Projeto

O projeto apresenta informações sobre a carreira em Ciência de Dados, incluindo:

- O que faz um cientista de dados  
- Tecnologias e ferramentas utilizadas  
- Tendências do mercado  
- Dicas para iniciantes  
- Relação com o ODS 9 (Indústria, Inovação e Infraestrutura)

Além disso, foi desenvolvido um site com design moderno (estilo hacker) utilizando HTML e CSS, publicado através do GitHub Pages.

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3  
- Git e GitHub

Este projeto demonstra como a Ciência de Dados contribui para a inovação, otimização de processos e desenvolvimento tecnológico, alinhando-se ao Objetivo de Desenvolvimento Sustentável 9.
